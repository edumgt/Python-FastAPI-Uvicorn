# Day 153 – 로보어드바이저 포트폴리오 최적화 실습

> **소요시간**: 8시간 | **Phase**: 8 확장 – 로보어드바이저 프로젝트

---

## 🎯 학습 목표

- 투자성향 기반 모델 포트폴리오 자동 추천
- 종목 스크리닝 + 자산배분 + 리밸런싱 통합
- 성과 리포트(수익률·변동성·MDD·샤프) 자동 생성
- 모의투자 의사결정 루프 설계

---

## 📖 이론 (08:00 – 10:00)

- 위험성향(보수/중립/공격)에 따른 제약 조건
- 전략 엔진: 스크리닝 → 최적화 → 검증 → 보고
- 운영 관점: 드리프트 감지, 재조정 트리거

3. 월간 리밸런싱 백테스트 + 보고서 생성
